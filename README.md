# StyleSnap - Digital Closet & Outfit Planner

> **A Progressive Web App for managing your wardrobe, getting AI-powered outfit suggestions, and sharing style ideas with friends.**

---

## üéØ Project Overview

StyleSnap is a production-ready PWA that helps users digitize their wardrobe, receive weather-based outfit suggestions, track wear history, and share outfit ideas with friends. Built with Vue.js 3, Supabase, and Cloudinary.

### Key Features
- üì± **PWA with Offline Support** - Installable, works offline, mobile-optimized
- üëî **200-Item Digital Closet** - Upload & organize clothing with categories, tags, and filters
- ü§ñ **AI-Powered Suggestions** - Weather-based outfit recommendations with learning
- üå§Ô∏è **Weather Integration** - Real-time weather data for smart outfit suggestions
- üë• **Social Features** - Friend system with outfit sharing and privacy controls
- üîî **Push Notifications** - Real-time alerts for friend requests, likes, comments
- üìä **Analytics & Insights** - Wear tracking, most-worn items, outfit collections
- üîê **Google OAuth** - Secure authentication with session management
- üìà **Performance Optimized** - Lazy loading, virtual scrolling, image optimization

- **18 standardized colors:** Consistent color matching for outfit algorithms
- **Simple categories:** 5 categories only (extensible in future)
- **Algorithm-based suggestions:** Outfit generation using permutations and rules (no external AI APIs)
